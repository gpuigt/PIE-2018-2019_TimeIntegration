Temporal Methods
================

The temporal method is a function used to solve the equation for a given right hand side:

.. math::
   \left\{\begin{aligned}
    \dot{y}\left(t\right) &= f\left(y, t\right) \\
    y\left(t_0\right) &= y_0
   \end{aligned}\right.
