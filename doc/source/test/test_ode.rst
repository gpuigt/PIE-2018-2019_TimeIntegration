Tests on ODE
============

The module ``pie.test.test_ode`` can be run to test some temporal methods on an ODE.

To use this module, add, remove, comment or uncomment any temporal method in ``METHODS``,
then run ``compare_methods`` with the desired :doc:`problem <problem_ode>`.
